## 简介

使用 Next.js + TypeScript + Tailwindcss + MDX + zustand 开发的静态博客

[在线预览](https://tsuizen.cn)

## 使用指南

克隆到本地

安装依赖

```bash
pnpm install
```

启动

```bash
pnpm run dev
```

## 功能

- [x] 使用 husky + eslint + tsc + lint-staged 规范代码
- [x] SSG
- [x] 响应式设计
- [x] 黑暗模式
- [x] MDX 自定义组件
- [x] 文章目录
- [x] 文章分类，文章标签，阅读时间统计
- [x] SEO & sitemap
- [x] RSS
- [x] Youtube, Bilibili 嵌入支持
- [ ] 添加动态功能，例如点赞、评论
