## 简介

使用Next.js + TypeScript + Tailwindcss + MDX + zustand 开发的静态博客

[在线预览](https://tsuizen.cn)

## 使用指南

克隆到本地

安装依赖
```
pnpm install
```
启动
```
pnpm run dev
```
## 功能
